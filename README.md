Listenify 🎧
Listenify is a modern music player built with Vanilla JavaScript.

The goal of this project is to practice building interactive UI and handling audio playback using pure JavaScript without frameworks.

✨ Features
Play and pause music
Next and previous track
Progress bar control
Track title and information display
Clean and simple user interface

🛠 Tech Stack
HTML
CSS
JavaScript (Vanilla)

🚀 Getting Started
Clone the repository
git clone https://github.com/yuseffrz/listenify.git

Open the project folder
cd listenify

Open index.html in your browser
That’s it. No installation required.
